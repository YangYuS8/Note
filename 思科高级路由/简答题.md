**1. BGP如何通过AS_PATH属性实现环路避免？请结合具体场景说明。**

- **防环机制**：检查AS_PATH中是否有自身AS号，若有则丢弃。
- **场景示例**：AS100向AS200通告路由（AS_PATH=100）。若AS200将其通告回AS100，AS100会看到AS_PATH=200,100（含自身AS号100），丢弃该路由。

**2. RSTP（快速生成树协议）相比STP有哪些改进？**
- **收敛更快**：通过“提议-同意”机制，秒级收敛。
- **端口角色**：新增**替代端口**（Alternate Port）和**备份端口**（Backup Port）。
- **拓扑变更**：边缘端口变化直接触发更新。

**3. DAI为何需要区分“信任端口”和“非信任端口”？信任端口通常指哪些接口？**
- **区分原因**：信任端口不检查ARP（性能优化），非信任端口严格检查（防止ARP欺骗）。
- **信任端口**：上行端口、合法服务器端口、手动标记为信任的端口。

**4. VLAN的核心作用是什么？如何划分VLAN？**

- **核心作用**：逻辑隔离广播域，提高安全性和管理灵活性。
- **划分方式**：基于端口、MAC地址、协议、子网。

**5. 静态路由的默认优先级（管理距离）是多少？动态路由协议的典型优先级如何比较？**
- **静态路由默认**：1（数值越小越优先）。
- **动态路由**：直连路由=0，OSPF=110，RIP=120。
- **结论**：静态路由通常优先于动态路由。

**6. DAI（Dynamic ARP Inspection）如何防止ARP欺骗攻击？其检查逻辑是什么？**
- **防护机制**：比对ARP报文的**源IP**和**源MAC**与DHCP Snooping绑定表。
- **检查逻辑**：匹配则放行，不匹配则丢弃并记录。
- **检查范围**：仅对**非信任端口**的ARP报文进行检查。

**7. 如何在Cisco路由器上配置一条到192.168.1.0/24网络的静态路由，下一跳为10.0.0.1？**
```cisco
  ip route 192.168.1.0 255.255.255.0 10.0.0.1
```

**8. IPSG（IP Source Guard）如何利用DHCP Snooping绑定表实现安全防护？**
- **工作原理**：检查数据包的**源IP**和**源MAC**是否与DHCP Snooping绑定表一致。
- **防护动作**：一致则放行；不一致则丢弃，防止IP/MAC欺骗。

**9. BGP（边界网关协议）的核心功能是什么？其设计目标与IGP（内部网关协议）有何本质区别？**
- **BGP核心功能**：跨**自治系统（AS）** 交换路由信息，实现策略化域间路由。
- **与IGP区别**：
  - **作用范围**：BGP用于AS间（域间），IGP用于AS内（域内）。
  - **设计目标**：BGP侧重于策略控制与可达性；IGP侧重于最优路径计算与快速收敛。

**10. 静态路由可能导致哪些问题？如何排查？**
- **常见问题**：下一跳不可达、路由黑洞、缺乏冗余。
- **排查**：`show ip route`检查路由表，`ping`/`traceroute`测试连通性，检查接口状态。

**11. 在静态IP环境中，DHCP Snooping绑定表如何维护？IPSG和DAI是否仍有效？**
- **绑定表维护**：需**手动添加**静态IP用户的IP-MAC-端口绑定条目。
- **有效性**：IPSG和DAI**仍然有效**，但依据变为手动维护的绑定表。

**12. STP端口状态有哪些？各状态的作用是什么？**

- **Blocking**：阻塞，防环，不转发。
- **Listening**：监听，参与拓扑计算，不转发。
- **Learning**：学习MAC，不转发。
- **Forwarding**：正常转发。
- **Disabled**：端口关闭。

**13. 静态路由与动态路由的核心区别是什么？**
- **静态路由**：管理员**手动配置**，不自动更新。
- **动态路由**：通过**路由协议**自动学习并适应拓扑变化。

**14. 动态路由协议RIP、OSPF、EIGRP分别基于什么算法？各有什么特点？**

- **RIP**：**距离向量算法**。特点：跳数限制15，收敛慢。
- **OSPF**：**链路状态算法（SPF）**。特点：分层设计，收敛快，无环路。
- **EIGRP**：**混合型算法（DUAL）**。特点：收敛极快，支持不等价负载均衡（思科私有）。

**15. 动态路由协议重分发（Route Redistribution）可能引发什么问题？如何解决？**
- **问题**：次优路径、路由环路、路由信息爆炸。
- **解决**：使用**route-map**过滤、**调整度量值**、避免**双向重分发**或使用**路由标记（Tag）**。

**16. MSTP（多生成树协议）如何解决STP/RSTP的局限性？**

- **局限性**：所有VLAN共享一棵树，链路利用率低。
- **解决方案**：支持**多实例（MST Instance）**，不同VLAN映射到不同实例，实现负载均衡。

**17. 动态路由协议中，水平分割（Split Horizon）和毒性逆转（Poison Reverse）如何防止路由环路？**

- **水平分割**：不从接收接口发回路由信息。
- **毒性逆转**：路由失效时，向邻居发送跳数无穷大（如16）的更新，强制删除。

**18. 若接入层用户报告网络时断时续，可能因DHCP Snooping绑定表异常导致？如何分析？**
- **原因**：绑定表IP/MAC冲突、表项过期、被篡改。
- **分析**：检查绑定表重复/错误条目、核对用户实际信息、查看交换机日志。

**19. DHCP Snooping在接入层安全中的核心作用是什么？其生成的绑定表包含哪些关键信息？**
- **核心作用**：防止DHCP欺骗攻击；为IPSG/DAI提供绑定表依据。
- **关键信息**：MAC地址、IP地址、VLAN、接入端口、租约时间。

**20. BGP的路由选路过程中，哪些属性会影响最终路径选择？请按优先级顺序列举并解释。**
- **优先级顺序**（前几位）：
  1. **Weight**：本地有效，越大越优。
  2. **LOCAL_PREF**：AS内部有效，越大越优。
  3. **AS_PATH**：越短越优。
  4. **MED**：越小越优，用于影响相邻AS入口。

**21. MSTP区域间如何实现互通？需要注意什么？**
- **互通机制**：通过**CIST**交换BPDU实现区域间连通。
- **注意事项**：VLAN与实例映射需一致；核心交换机建议设为CIST根桥；确保设备支持MSTP。

**22. 若攻击者同时伪造DHCP响应和ARP报文，DHCP Snooping、IPSG、DAI如何联合防御？**
- **联合防御**：
  1. **DHCP Snooping**：过滤伪造DHCP响应，生成正确绑定表。
  2. **IPSG**：依据绑定表检查数据包源IP/MAC。
  3. **DAI**：依据绑定表检查ARP报文。
- **效果**：三者协同实现从IP分配到数据转发的全链路防护。

**23. 在VLAN环境中，DHCP Snooping绑定表的作用范围是什么？跨VLAN通信是否受影响？**
- **作用范围**：以**VLAN为单位**独立维护。
- **对跨VLAN通信**：无直接影响，跨VLAN通信由三层设备处理。

**24. STP（生成树协议）的核心目的是什么？它如何防止二层环路？**
- **核心目的**：在冗余链路中阻塞特定端口，构建无环路树状拓扑，防止广播风暴。
- **防环机制**：选举根桥、确定根端口和指定端口、阻塞其他端口。

**25. 某企业网络同时使用静态路由和OSPF，如何确保静态路由优先于OSPF路由？**
- **方法**：设置静态路由的管理距离低于OSPF（如设为100）：
  ```cisco
  ip route 目标网络 掩码 下一跳 100
  ```

**26. 静态路由的浮动静态路由（Floating Static Route）有什么作用？如何配置？**
- **作用**：作为主路由的备份路径，主路由失效时自动启用。
- **配置**：配置相同目标网络但管理距离更大的静态路由（如150）：
  ```cisco
  ip route 目标网络 掩码 备份下一跳 150
  ```

**27. 如何在Cisco交换机上配置基于端口的VLAN？**
```cisco
  vlan 10
    name SALES
  interface Gi0/1
    switchport mode access
    switchport access vlan 10
```

**28. 什么是PVLAN（私有VLAN）？它包含哪些端口类型？**
- **定义**：在主VLAN内划分从VLAN，实现同一网段主机的二层隔离。
- **端口类型**：**混杂端口**（可通所有从VLAN）、**隔离端口**（仅通混杂端口）、**团体端口**（团体内互通且通混杂端口）。

**29. 某网络中STP拓扑频繁变化，可能的原因及解决方案是什么？**
- **原因**：物理链路不稳定、BPDU丢失、非法设备接入。
- **解决**：检查物理链路；启用**BPDU Guard**和**Root Guard**。

**30. 如何在Cisco交换机上配置MSTP？**
```cisco
  spanning-tree mode mst
  spanning-tree mst configuration
    instance 1 vlan 10,20
    name REGION1
    revision 1
```

**31. 在OSPF中，如何将一个接口划入特定区域（如Area 1）？**
```cisco
  interface Gi0/0
    ip ospf 1 area 1
```

**32. IPSG误阻断合法流量时，可能的原因有哪些？如何解决？**
- **原因**：绑定表过时、静态IP未绑定、误在信任端口启用IPSG。
- **解决**：更新绑定表、为静态IP手动绑定、检查端口信任状态。
